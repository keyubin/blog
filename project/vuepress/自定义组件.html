# 自定义组件

在.vuepress文件夹下创建components文件夹,文件夹内可以放各种vue组件，框架将自动将他们作为全局组件使用。

# 添加element-ui

  • 在.vuepress文件夹下添加enhanceApp.js

  • 这将会自动将element-ui配置至全局，任何md文件中都可以使用element-ui

import Element from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'

export default ({
  Vue, 
}) => {
  // ...做一些其他的应用级别的优化
  Vue.use(Element)
}
